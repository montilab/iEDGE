do_gistic2ge wrapper for performing differential expression test

Usage

do_gistic2ge(ge, gistic, case = c(1, 2), control = (0), interaction_type = "cis", test = "limma", alternative = "equal", boundary = "peak", logged = TRUE)

Arguments

ge
internal ge object
gistic
internal gistic object
case
values to be considered altered samples in gistic lesions file, default c(1,2)
control
values to be considered unaltered samples in gstic lesions file, default c(0)
interaction_type
"cis" or "trans" type of analysis to performed
test
"limma" or "ttest" DE test to be used, default "limma"
alternative
"equal", "greater", or "less" is DE test one-sided? rule of thumb for trans alternative = "greater", for cis, alterative = "greater" for amp and "less" for del
logged
TRUE if gene expression is log2-transformed, DE is performed on log2 expression values

Value

data frame with DE results

Description
