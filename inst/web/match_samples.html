match_samples match samples found in ge and gistic objects

Usage

match_samples(gistic, ge, fmap.file, fmap.gistic.col = "snp.array.ID", fmap.ge.col = "mrna.array.ID", ge.confounder.file = NA)

Arguments

gistic
gistic object
ge
ge object
fmap.file
file containing mapping from ge to gistic samples
fmap.gistic.col
column name in fmap.file containing gistic ids
fmap.ge.col
column name in fmap.file containing ge ids
ge.confounder
sample confounder indicator file name, default = NA, no confounder

Description

match_samples match samples found in ge and gistic objects
