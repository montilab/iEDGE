base.summary adds additional columns to DE summary table from gene expression data

Usage

base.summary(ge.mat, groups, groups.label = c("case", "control"), logged = TRUE)

Arguments

ge.mat
gene expression matrix
groups
sample indicator for groups
groups.label
c(a, b) a = label for positive samples, b = label for negative samples
logged
TRUE if gene expression is log2-transformed, base.summary provides statistics on unlogged data

Description

base.summary adds additional columns to DE summary table from gene expression data
